<task type="auto">
  <name>Create appliance detector module</name>
  <files>src/appliance_detector.py</files>
  <action>
    Create src/appliance_detector.py with:
    - ApplianceDetector class that detects appliances in frames
    - ApplianceType enum: PROJECTOR, MONITOR, LIGHT, CEILING_FAN, WALL_FAN, UNKNOWN
    - Status enum: ON, OFF, UNKNOWN
    
    - detect_appliance(frame, roi) -> ApplianceType
      - Use brightness pattern analysis and shape features
      - Projector: bright white/blue light in center, lens glow
      - Monitor: rectangular screen with bright center, edges darker
      - Light: bright circular/rectangular area, strong glow
      - Ceiling fan: circular blades, oscillating pattern (distinguishing feature)
      - Wall fan: rectangular frame with blades
    
    - classify_status(frame, appliance_type, roi) -> Status
      - Calculate mean brightness within ROI
      - Calculate brightness variance
      - ON threshold: mean_brightness > 100 OR high variance (screen flicker)
      - OFF: brightness < 50
      - Use edge detection to identify screen/fan vs light
    
    - Key distinction for fans vs other appliances:
      - Fans have oscillating brightness patterns over time
      - Fans have blade-like edges (detected via Canny edge detection)
      - Projectors/monitors have more uniform screen glow
    - get_roi(frame, appliance_type) -> bounding_box
      - Define typical ROIs for each appliance type based on frame position
  </action>
  <verify>python -c "from src.appliance_detector import ApplianceDetector, ApplianceType, Status; print('Appliance detector OK')"</verify>
  <done>Appliance detector module created with detection and classification functions</done>
